

# Prueba proyecto trabajo regresión logística en R

## Uso git y github

Modelo básico de regresión logística usando el dataset de mtcars. La variable objetivo es si el 
coche es manual o automático a partir de su peso, la potencia y el consumo.

Uso de `umbral` de 0.5 para tomar la decisión de si (autónomo = 0, manual = 1)
